# 栈

- 栈是抽象的概念，栈可以用「数组」或者「链表」实现。
  - 使用「数组」实现的栈，删除和添加元素发生在数组的末尾；
  - 使用「链表」实现的栈，删除和添加元素发生在数组的开头；
